# Sistema de Restaurante Pesqueiro 
 
## Estrutura do Projeto 
 
- `backend/`: API Node.js com Express e PostgreSQL 
- `frontend/`: Interface web em HTML, CSS e JavaScript 
 
## Iniciar o Projeto 
 
### Backend 
 
```bash 
cd backend 
npm install 
npm run dev 
```
